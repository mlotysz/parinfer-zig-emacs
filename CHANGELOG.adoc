= Changelog

== 0.5.0

Rewrite from Rust to Zig. Emacs-only, Clojure-only.

=== Changed

* Entire codebase rewritten from Rust to Zig (0.14+)
* Removed all non-Emacs editor support (Vim, Neovim, Kakoune, CLI, WebAssembly)
* Removed all non-Clojure language support (Lisp, Scheme, Guile, Janet, Racket, Yuck, Fennel, Dune)
* Removed language configuration options (`comment_char`, `string_delimiters`, `lisp_vline_symbols`, `lisp_block_comments`, `guile_block_comments`, `scheme_sexp_comments`, `janet_long_strings`)
* Removed Nix build system, Cargo, npm packaging
* Pure-Zig Emacs module API bindings (no C header or `emacs` crate dependency)
* Build system: `zig build` replaces `cargo build --release`

=== Added

* Emacs integration test suite (`zig build test-emacs`)
